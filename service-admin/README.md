# ministryofjustice/opg-lpa-admin

## Pre-requisites

PHP 7.4

Composer

## Installation

git clone the project then:

```
cd service-admin
composer install
```

## Tests

Run the unit tests from the `service-admin` directory (after running composer)
with:

```
php vendor/bin/phpunit
```
