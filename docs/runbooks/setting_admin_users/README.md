# Setting admin users

## instructions

Access to the admin service is now controlled by AWS Cognito using EntraID provider. Anyone who can successfully sign in via the Cognito hosted UI (i.e. is a member of the  relevant Cognito user pool) is granted access to the admin service.

To grant a user access to the admin service see this guide, [Granting Access to Make Admin](https://opgtransform.atlassian.net/wiki/x/BYDtIAE?xpis=eyJicmlkZ2UiOiJzbWFydExpbmtzIiwiaWQiOiIxNzg0Nzk4MTY0MzEwIiwic291cmNlIjoiamlyYSJ9)
